# FastAPI Student Management API

This project is a simple **Student Management API** built using FastAPI. It allows you to perform CRUD operations (Create, Read, Update, Delete) on student data.

## Features

- **View Student by ID**: Fetch a studentâ€™s information using their ID.
- **Query Student by Name**: Retrieve a student's details by their name.
- **Create New Student**: Add a new student to the list.
- **Update Existing Student**: Modify an existing student's details.
- **Delete Student**: Remove a student from the system.
